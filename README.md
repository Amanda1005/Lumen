# Lumen

**Bloomberg-style trust ratings for on-chain AI agents.**

Lumen rates every AI agent on Mantle's ERC-8004 registry across 5 dimensions, detects sybil attacks, and publishes verifiable scores on-chain — so investors, dApps, and users can finally tell real agents from clones.

Built for [The Turing Test Hackathon 2026](https://dorahacks.io/hackathon/mantleturingtesthackathon2026).

---

## The Problem

Mantle gave AI agents on-chain credentials. But the trust layer is being gamed:

- **91.8%** of agents on Mantle ERC-8004 are sybil clones from a single wallet
- **99%** of reputation signals come from one self-feedback wallet
- **Zero** independent rating infrastructure exists

Investors are flying blind. Lumen is the trust infrastructure they need.

---

## What Lumen Does

Lumen scans every agent on Mantle ERC-8004 and produces:

1. **Lumen Score (0–100)** across 5 dimensions
2. **Risk Status** (Safe / Suspicious / Sybil)
3. **Institutional analyst commentary** generated by Claude
4. **On-chain verifiable scores** via `LumenScoringRegistry`
5. **Real-time Telegram alerts** for new sybil detections

### Scoring Model

| Dimension | Weight | What it measures |
|-----------|--------|------------------|
| Completeness | 25% | Metadata quality (name, description, endpoints) |
| Capability | 25% | Skills, tools, integration depth |
| Owner Reputation | 20% | Sybil detection, wallet patterns |
| Verifiability | 15% | External proofs, schema compliance |
| Activity | 15% | On-chain feedback (penalized for self-feedback) |

## Run Locally

```bash
# Backend
cd backend
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # add your ANTHROPIC_API_KEY
python scripts/pipeline.py   # index + score all agents
uvicorn app.main:app --reload

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```

Open http://localhost:3000
